# Contributing to Otto-task
First off, thanks for taking the time to contribute, Architect!

### How to Contribute
1. **Fork the Bunker**: Create your own copy of the repository.
2. **Branching**: Use `feat/` for new systems or `fix/` for bug resolution.
3. **Code Style**: Maintain the **Pure Neon Tech** style in all CSS/HTML files.
4. **Pull Requests**: Explain how your change improves the Shadow Agent's intelligence.

### Guidelines
- Always use environment variables for sensitive keys.
- Test your deployment in a mobile environment (Termux) before submitting.
