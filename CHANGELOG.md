# Changelog
All notable changes to the **Otto-task** project will be documented in this file.

## [1.0.0-Beta] - 2026-03-02
### Added
- **Shadow Agent Core**: Integration with Gemini 2.0 Flash for autonomous negotiations.
- **Cyberpunk HUD**: Responsive web interface with real-time scan-line effects.
- **Cloudinary Sync**: Automatic high-speed image delivery system for the market.
- **Termux Support**: Environment patch to allow mobile deployment via Android.

### Changed
- Migrated database architecture from SQLite to **Neon PostgreSQL** for production.
- Refined the "Agentic Negotiator" logic for better price margin protection.

### Fixed
- Fixed the "Particle Button" hang-up issue by optimizing CSS overflow.
- Resolved Django File Locks error (Error 38) in mobile environments.
